# 排序与分页
## 1.排序
```SQL
select * from employee;
#按照id从低到高列出成员信息
#使用order by对查询到的数据进行排序操作
#升序：ASC(默认)
#降序：DESC
select id as 'ID',name as '姓名'
from employee
ORDER BY id ASC;
```
## 2.二级排序
```SQL
select id as 'ID',name as '姓名'
from employee
ORDER BY id ASC /*,XXX ASC*/;
```
# 分页
```SQL
#使用关键字limit实现数据的分页操作
#每一页显示3条记录，此时显示第1页
select id as 'ID',name as '姓名'
from employee
limit 0,3;#偏移量0，3条数据(偏移量为0表示从第一条数据开始)
#每一页显示3条记录，此时显示第2页
select id as 'ID',name as '姓名'
from employee
limit 3,3;#到第X页的公式为：偏移量=每页显示的数据量*(X-1)

```
